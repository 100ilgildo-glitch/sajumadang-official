# 사주마당 - 평생사주 및 운세 상담 웹사이트

전문 명리학 플랫폼과 함께하는 정확하고 상세한 사주 분석 서비스를 제공하는 웹사이트입니다.

## 🌟 프로젝트 개요

**사주마당**은 고객이 온라인으로 편리하게 사주 상담을 신청할 수 있는 전문 웹사이트입니다. 명리학 전문 플랫폼과 협업하여 24시간 내 이메일로 상세한 PDF 리포트를 제공합니다.

### 주요 특징
- ✨ 전통과 현대가 조화된 따뜻한 디자인
- 📱 완벽한 반응형 디자인 (모바일, 태블릿, 데스크톱)
- 🎯 직관적인 사용자 경험
- 💰 실시간 금액 자동 계산
- ✅ 스마트 폼 검증 시스템
- 🔒 개인정보 보호

## 📋 제공 서비스

1. **평생사주** (1인 38,000원 / 2인 60,000원)
   - 타고난 성향과 특성 분석
   - 인생의 중요한 시기와 전환점
   - 종합 운세 분석

2. **2026년 신년운** (1인 25,000원 / 2인 38,000원)
   - 새해 운세 및 흐름 분석
   - 한 해 성공 가이드

3. **재물운** (1인 25,000원 / 2인 38,000원)
   - 재물의 흐름과 투자 시기
   - 재테크 방향성 제시

4. **건강운** (1인 20,000원 / 2인 30,000원)
   - 신체적·정신적 건강 분석
   - 주의 시기 및 건강 관리 방법

5. **직업운** (1인 20,000원 / 2인 30,000원)
   - 적성과 재능 분석
   - 직업 선택 및 승진/이직 시기

6. **궁합** (2인 50,000원)
   - 연애·결혼·사업 파트너 궁합
   - 관계 유지 조언

  - 성별
  - 생년월일 종류 (양력/음력)
  - 생년월일
  - 태어난 시간 (선택사항)
